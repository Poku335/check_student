# ระบบเช็คชื่อเข้าเรียน

## การติดตั้ง

### 1. Backend
```bash
cd attendance-backend
npm install
npm start
```

### 2. Frontend
```bash
cd attendance
npm install
npm run dev
```

## การใช้งาน

1. เปิด http://localhost:5173
2. เข้าสู่ระบบด้วย:
   - Username: `admin`
   - Password: `password123`
