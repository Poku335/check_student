### Backend
```bash
cd attendance-backend
npm install
npm start
```

### Frontend
```bash
cd attendance
npm install
npm run dev
```

## Default Login
- Username: `admin`
- Password: `password123`

## Environment Variables
```
DB_PATH=./database.sqlite
JWT_SECRET=your-secret-key-here
PORT=3000
```

- Backend: http://localhost:3000
